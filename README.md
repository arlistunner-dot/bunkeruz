# So‘nggi Joy

Telegram Mini App ichida ishlaydigan kartali bunker uslubidagi online stol o‘yini.

## Tuzilma

- backend: API, real-time o‘yin serveri, xona va karta logikasi
- frontend: Telegram Mini App interfeysi
- keyinroq: Telegram bot ulanishi

## Birinchi MVP maqsadi

1. Xona yaratish
2. Kod orqali xonaga kirish
3. O‘yinchilar ro‘yxati
4. Kartalarni tarqatish
5. Raundma-raund karta ochish
6. Ovoz berish
7. Natijani chiqarish
